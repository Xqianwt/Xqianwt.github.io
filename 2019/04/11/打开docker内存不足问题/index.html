打开docker内存不足问题

因为远古电脑内存只有4g，在使用scrapy-splash爬虫时要用到docker，安装好打开docker—desktop之后，提示Not enough memory to start Docker Desktop，明显是电脑内存不足咯！

找到的解决方法使用内存管理工具：memory cleaner。
